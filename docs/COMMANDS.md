# VoiceToPlay 语音命令列表

## 战斗相关

### 出牌

| 关键词      | 说明                   |
|----------|----------------------|
| `{卡牌名}`  | 如 "防御"、"打击"，总是选择最左边的 |
| `{卡牌名}一` | 第1张同名卡，如 "防御一"       |
| `{卡牌名}二` | 第2张同名卡               |

**场景**: 战斗中

---

### 选择敌人

| 关键词             | 说明      |
|-----------------|---------|
| `第一个` / `瞄准第一个` | 瞄准第1个敌人 |
| `第二个` / `瞄准第二个` | 瞄准第2个敌人 |
| ...             | 支持到第九个  |

**场景**: 战斗中

---

### 结束回合

| 关键词    |
|--------|
| `结束`   |
| `结束回合` |

**场景**: 战斗中

---

## 卡牌选择

### 牌堆选牌（消耗/升级等）

| 关键词           | 说明    |
|---------------|-------|
| `{卡牌名}`       | 选择该卡牌 |
| `第一张` ~ `第九张` | 按位置选择 |
| `向上滚` / `上滚`  | 向上滚动  |
| `向下滚` / `下滚`  | 向下滚动  |

**场景**: 卡牌网格选择界面

---

### 卡牌奖励选择

| 关键词           | 说明    |
|---------------|-------|
| `{卡牌名}`       | 选择该卡牌 |
| `第一张` ~ `第九张` | 按位置选择 |
| `跳过`          | 跳过选择  |

**场景**: 卡牌奖励界面

---

## 地图与导航

### 地图路径

| 关键词           | 说明          |
|---------------|-------------|
| `左边`          | 最左边的路径      |
| `右边`          | 最右边的路径      |
| `中间`          | 中间路径（仅3条路时） |
| `第一条` ~ `第九条` | 按序号选择       |
| `上去`          | 最左边的路径      |

**场景**: 地图界面

---

### 前进按钮

| 关键词  |
|------|
| `前进` |

**场景**: 有前进按钮时

---

### 确认按钮

| 关键词  |
|------|
| `确定` |
| `确认` |

**场景**: 有确认按钮时

---

## 房间与事件

### 事件选项

| 关键词             | 说明       |
|-----------------|----------|
| `第一个` / `第一个选项` | 第1个选项    |
| `第二个` / `第二个选项` | 第2个选项    |
| ...             | 支持到第九个   |
| `继续`            | 第一个选项的别名 |

**场景**: 事件房间

---

### 火堆选项

| 关键词  | 说明     |
|------|--------|
| `休息` | 休息恢复生命 |
| `锻造` | 升级卡牌   |
| `挖掘` | 获得遗物   |

**场景**: 火堆房间

---

### 宝箱遗物

| 关键词           | 说明      |
|---------------|---------|
| `打开宝箱`        | 打开宝箱    |
| `查看遗物`        | 查看当前遗物  |
| `下一个遗物`       | 切换到下一个  |
| `选择遗物` / `拿取` | 选择当前遗物  |
| `{遗物名}`       | 直接选择该遗物 |

**场景**: 宝箱房间

---

### 奖励领取

| 关键词           | 说明     |
|---------------|--------|
| `金币`          | 领取金币   |
| `卡牌一` / `卡牌二` | 打开卡牌奖励 |
| `{遗物名}`       | 领取该遗物  |
| `{药水名}`       | 领取该药水  |

**场景**: 奖励界面

---

## 主菜单

### 菜单按钮

| 关键词      | 说明                   |
|----------|----------------------|
| `{按钮文本}` | 如 "开始游戏"、"继续游戏"、"设置" |

**场景**: 主菜单

---

## 词表类型

| 类型 | 命令                            |
|----|-------------------------------|
| 静态 | 结束回合、前进、确认                    |
| 动态 | 出牌、选敌人、选卡牌、地图、事件、火堆、宝箱、奖励、主菜单 |
